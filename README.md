TradeAgent
==========

Route Trade Agent (ASP.NET MVC 4 + OpenStreetMap + OpenLayer + AJAX)

Реализация Алгоритма Дейкстры с применением карт от OSM, при помощи JS библиотеки OpenLayer.
Был взят за основу алгоритм с Хабра http://habrahabr.ru/post/63347/  , проанализирован, исправлен и дополнен.
Добиться идеального вычитывания пока все же не получилось, хоть и удалось сократились коллизия в несколько раз.
Плюс капелька адаптивности.
